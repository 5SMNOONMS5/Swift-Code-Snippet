#Code Snippet
===============================

My personal code-snippet sync tool, store in one place, using everywhere.

## Quick Start

* Git clone first.
* Run the script, this will copy all the files into your Xcode load path.

```bash
git clone git@github.com:5SMNOONMS5/XcodeSyncTools.git
cd XcodeSyncTools
sh xcode-sync.sh -s
```

## More detail

```bash
sh xcode-sync.sh -h
```
